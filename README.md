Descripción
Esta es una aplicación de e-commerce simple construida utilizando REACT. La aplicación cuenta con una barra de navegación responsiva, un widget de carrito de compras y un contenedor de mensajes de bienvenida.

Características
Barra de Navegación Responsiva: Una barra de navegación construida con BootStap con enlaces a diferentes secciones de la aplicación.

Widget de Carrito de Compras: Muestra un ícono de carrito de compras con un conteo de ítems hardcodeado.

Mensaje de Bienvenida: Un contenedor que muestra un mensaje de bienvenida personalizable.
